?                          ?? TROUBLESHOOTING                                  ?
????????????????????????????????????????????????????????????????????????????????

   Problem: "dotnet is not recognized"
   ????????????????????????????????????????????????????????????
   Solution: Install .NET 8.0 SDK
   ? https://dotnet.microsoft.com/download/dotnet/8.0
   ? Restart terminal after installation


   Problem: Build fails with errors
   ????????????????????????????????????????????????????????????
   Solution: 
   ? Close all StudyBuddy.exe instances
   ? Delete StudyBuddy\bin and StudyBuddy\obj folders
   ? Run build again


   Problem: File is too large
   ????????????????????????????????????????????????????????????
   This is normal!
   ? Includes .NET 8 runtime (~60 MB)
   ? Includes all dependencies
   ? Users don't need to install anything
   ? Trade-off for portability


   Problem: Antivirus blocks the .exe
   ????????????????????????????????????????????????????????????
   Common with unsigned executables:
   ? Add exception in antivirus
   ? For production: Consider code signing
   ? Tell users it's safe to run
